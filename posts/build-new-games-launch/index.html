Build New Games Launch

tutorial web applications
November 22nd, 2022

Today we launched Build New Games on the main stage at JSConf with our friends from the Microsoft IE Team.

Build New Games is a new resource for developers who want to build and release cross browser and cross device Open Web games.

Over the next several months, we will be working with the throngs of programmers who have made the leap into Open Web game development to create a compendium of game development knowledge.

Topics will include mobile games, html5 technology, physics, rendering engines, game design for the open web, input devices, and more!

This site comes as a natural progression from our work with the Gamelab, and Bocoup’s New Game Conference.

If you’re interested in writing for Build New Games, get in touch with Darius Kazemi (darius@bocoup.com) with your idea.
